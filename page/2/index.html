    Batch

    Batch 批量处理相关。

    %~dp0[获取当前路径]

    1. %~dp0 “d”为Drive的缩写，即为驱动器，磁盘、“p”为Path缩写，即为路径，目录
    2. cd %~dp0 ：进入批处理所在目录
    3. cd %~dp0bin\ ：进入批处理所在目录的bin目录
