Jupyter 主题
# 安装 Jupyter-theme
命令
:	```python
	pip install jupyterthemes
	```

更换主题
:	```python
	jt -l # 列出主题
	jt -t themeName # 更换主题
	jt -r # 恢复默认主题
	```
